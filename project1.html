Student Record System

A complete C-based system for managing student records, attendance, and grades efficiently.

Overview

The Student Record System is a console-based application written in the C programming language. It allows users to input, manage, and organize student data while providing automated sorting, attendance tracking, and grade computation features. The system uses structured programming principles and validation to ensure accurate and efficient record handling.
